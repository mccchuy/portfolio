ZOOX is a company creating self-driving or autonomous vehicles designed to make personal transportation safer, cleaner, and more enjoyable. The rider simply pays for the service, and the vehicle comes to the user, similar to rideshare services such as Uber or Lyft.

Prompt

The task was to design a conceptual UI for the in-ride experience for ZOOX vehicles. The system should be designed to be fully integrated into the vehicle. The user should be able to control anything a driver could typically control in a car. Functionalities include:

  • Welcome the user to the vehicle and have confirmation system for security
  • Display trip progress
  • Real time CarView
  • Contact support
  • Interior climate control adjustment
  • Different preset climate modes
  • Enable destination changes
  • Ability to save climate preferences and access them in future ride

Low-Fidelity Screens

Dashboard

The dashboard page is the main page, which displays a panic (!) button that will be present throughout the user experience. The main page is meant to mimic a typical car dashboard, with key features like the navigation and speedometer being present.

Challenges: The use of both the spedometer and the MPH reading at the top center are reduntant.

Navigation

The navigation page displays key information for the user, such as the date and time, distance until the next turn, estimated distance remaining, a trip progress bar, and a compass. It also includes several interactive features, such as a "Change Route" button, an expand button, and the ability to expand between three different map views: the standard navigation map, RealView, and CarView.

Challenges: The ability to change route is not easily accessible.

Climate 1

The first climate page displays different ways the user can adjust the climate settings in their ZOOX vehicle, including temperature, lighting, and preset climate modes for user convenience. It also includes the ability to save preferred controls.

Challenges: Cluttered; too much going on in one page, should be broken up into several pages. Unclear what the star symbol on the left is for

Climate 2

The second climate page displays the seat controls for the user to adjust, including the ability to activate and control seat warmers,

Challenges: Unclear how to get to this page from the original climate control page.

High-Fidelity Screens

Onboarding

In the initial screen, users are asked to confirm their identity by entering a code on their phone sent to them by the ZOOX app. The user has the option to resend a new code if they no longer can access their first code.

Dashboard

These two screens show the interaction of the panic button, which allows the user to call for help in the event of an emergency.

On the dashboard screen, the user is able to view information displayed in bright purple and interact with features in blue. This is true for all screens within the ZOOX UI.

The Dashboard screen is modeled after a a typical car dashboard, with the user having access to turn signal info, which mode the car is in, the spedometer, and the positioning of the wheel in real time.

It also gives the user the ability to view trip details, current date, time, and climate information and a navigation map.

Navigation

These two screens show the ability to expand the navigation view within the navigation page

The user also has the option to edit their trip on this page, switch between RealView (a window view) and CarView (a view of what ZOOX sees as it drives).

Climate Controls - Modes

These screens show the ability to choose between 3 preset modes, access access saved modes, and create custom modes.

The second screen shows the ability to save current climate control settings to be accessed later.

Climate Controls - Temperature

This screen shows the ability to control the temperature in the ZOOX vehicle through a dial, with the ability to control fan speeds on both sides, and the current temperature outside to better inform the user's temperature selection.

Climate Controls - Seats

This screen allows the user to adjust the seat by reclining or bringing the seat up, and bringing the seat forward or backward, and to adjust the seat warmer temperature.

Climate Controls - Lighting

This screen allows the user to control the lighting by either turning the lights on/off, dimming them and adjusting their color, and allowing in natural light by adjusting the blinds level and direction. The current weather is also made available to better inform the user's decision.
